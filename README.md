# SDETCourseFirstTask
This is a project for the first task in a SDET course

## Technology stack

1. Java
2. Maven
3. TestNG
4. Selenium WebDriver
5. IntelliJ
6. Java Faker
7. Allure


## Test execution

For test execution need to run this command in terminal:

```powershell
    
    mvn clean test
    
 ```

For generating and view allure report run:

```powershell
    
    mvn allure:serve
    
```
